# 伪装新闻聊天APK - 快速部署指南

## 🚀 5分钟快速部署

### 步骤1：准备GitHub仓库
1. 访问 https://github.com 注册账号（如已有账号请登录）
2. 点击右上角"+" → "New repository"
3. 输入仓库名：`stealth-news-chat`
4. 选择"Public"（公开，免费）
5. 勾选"Add a README file"
6. 点击"Create repository"

### 步骤2：上传项目文件
1. 在仓库页面点击"Add file" → "Upload files"
2. 将整个项目文件夹拖拽到上传区域
3. 点击"Commit changes"

### 步骤3：触发构建
1. 点击顶部"Actions"标签
2. **重要**：如果看不到工作流，请等待几分钟或刷新页面
3. 在左侧找到"Build Android APK"工作流（可能需要滚动查找）
4. 点击"Run workflow" → "Run workflow"
5. 等待约5-10分钟构建完成

### 步骤4：下载APK
1. 构建完成后，点击最新的工作流运行
2. 在"Artifacts"部分下载`stealth-news-chat-apk`
3. 将APK文件传输到安卓手机安装

## 📱 安装测试

### 安卓手机设置
1. **允许未知来源应用**
   - 设置 → 安全 → 未知来源 → 允许
   - 或设置 → 应用 → 特殊应用权限 → 安装未知应用

2. **安装APK**
   - 使用文件管理器找到下载的APK
   - 点击安装 → 接受权限 → 完成

## 🔧 常见问题解决

### 构建失败
- 检查项目文件是否完整上传
- 确保`build.gradle`文件正确
- 查看GitHub Actions日志找错误原因

### 安装失败
- 确保允许"未知来源"安装
- 检查Android版本兼容性（最低Android 5.0）
- 尝试清除安装包缓存后重试

### 功能异常
- 重启应用尝试
- 检查手机权限设置
- 反馈具体问题以便优化
